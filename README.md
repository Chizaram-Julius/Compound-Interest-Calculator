## **Compound Interest Calculator**

A simple and responsive web app that calculates compound interest based on user input (principal, interest rate, and number of years). Built with HTML, CSS, and JavaScript, this project demonstrates handling form inputs, validating values, and updating the UI dynamically.

### **Features**

* Calculates compound interest instantly
* Input validation for clean and error-free results
* Responsive and clean UI design
* Displays formatted final amount using `toLocaleString()`

### **Tech Stack**

* **HTML5** – structure
* **CSS3** – styling and layout
* **JavaScript** – logic & DOM manipulation

### **What I Learned**

* Using JavaScript to handle form data
* Applying input validation
* Updating the DOM dynamically
* Formatting currency using `toLocaleString()`
